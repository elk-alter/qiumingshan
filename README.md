## 功能列表

- App客户端
  - 注册登录
  - 答题（错题查看）
  - 成绩记录
- 网页后台
  - 题库编辑
  - 学生管理
- 网页客户端
  - 注册登录
  - 科目1-4知识学习
  - 科目1、4测试
  - 错题查看
  - 驾考圈社交功能（微型论坛）
